### 🚀 Featured Projects

#### 1. AI-Powered Learning Support Platform
*A comprehensive web service utilizing LLMs to generate and support learning content.*
- **Key Features:** Text analysis, automatic Question Generation (QG), and document-based Q&A.
- **Tech Stack:** - **Frontend:** Next.js, Tailwind CSS, shadcn/ui
  - **Backend:** FastAPI
  - **AI/NLP:** PyTorch, sense2vec, spaCy
  - **LLM:** Meta-Llama-3-8B (via LM Studio), T5-base-qg-hl (Hugging Face)
- **Role:** Full-stack Development & AI Pipeline (Solo Project)
- **Links:** [Frontend Repo](https://github.com/KieuLong1401/learning_support) | [Server Repo](https://github.com/KieuLong1401/learning_support_server) | [📺 Demo Video](https://drive.google.com/file/d/1iHHiRmnOtwTtCMt7iNbsDv572Siy9Kej/view?usp=sharing)

#### 2. Smart Parking Gate System (LPR)
*Real-time license plate recognition and barrier control system using YOLOv8.*
- **Tech Stack:** - **Embedded:** Raspberry Pi (GPIO), Arduino (C++)
  - **Backend:** FastAPI
  - **AI Model:** YOLOv8 (Fine-tuned for license plates), EasyOCR
- **Role:** - Developed Raspberry Pi & Arduino control logic.
  - Performed YOLOv8 model Fine-tuning and performance optimization.
- **Links:** [Hardware & Code Repo](https://github.com/KieuLong1401/auto_door) | [📈 Fine-tuning Process](https://drive.google.com/file/d/19zOTvGdlc1yCj9S2DEKbB3_p2YIjXSz/view?usp=sharing) | [📺 Testing Video](https://drive.google.com/file/d/13TebVtosPF3sOrBXkqPtoJollbXca6I/view?usp=sharing)
